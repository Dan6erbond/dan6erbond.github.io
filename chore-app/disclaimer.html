Usage of User FilesðŸ”—

Any files captured within the app can be uploaded to the application server to enable a persistent user experience across multiple devices. The developer (Dan6erbond) and publisher (TechDog) do not sell or offer these files to third-parties, nor are they used outside of the application scope to enable a seamless user experience. Images captured by the user may be processed by Google's Firebase servers and machine learning algorithms as well as locally ran logic to enable the capturing of QR codes and similar.
