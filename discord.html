Creating a server🔗

Creating a server is easy! Use this big button to get started and then follow Discord's official tutorial to create a basic server. Make sure you select a server region that's close to where you are to reduce lag and ping times:

Starting creating a server with this button which can be found below your server-list.

After that you'll be greeted by a server with two channels: #general and #voice

Now that you have your very own server, it's time to customize it. Follow the steps below to get the server exactly where you want it!

Setting up channels and categories🔗

Channels and categories are the most important part about a server. They'll help you keep your server organized and separate topics and most importantly, allow you to hide topics based on permissions.

To create a channel or category, right-click on the sidebar of your server and select the one you wish to create:

Right-click on your server's sidebar to create a channel or category.

Creating a channel will ask you if you want to create a text- or voice-channel. As you'd expect, text-channels don't allow users to connect with each other over Discord's voice-chat and vice-versa.

When creating either of the two, you additionally get the option to make it 'private'. Selecting that option will show you a list of roles for which the channel/category will then be visible. Everyone else won't have read/send access to the newly created item:

If you're unhappy with the position of the newly created channel or category, you can hold and drag it around with your mouse and drop it to the desired position.

Categories🔗

Categories are like containers for your text- and voice-channels. They allow you to group relevant ones together and setup permissions that will be taken over by the category's channels unless you wish to override them.

Dragging a channel below a category will automatically make it a child of the category and if the permissions of the channel don't align with those of the category, Discord will offer you to 'sync' them which means to take over those permissions of the category. If the permissions aren't synced, the original permissions of the channel will be kept.

Channels🔗

Text Channels🔗

Text channels are where users can post, well, text! Hovering over a text-channel will reveal a tooth-weel that allows you to configure it:

Hovering over a channel allows you to enter its settings.

There are a few key settings you'll want to look at and configure to your liking:

  • Topic: This is basically a channel's description. It supports regular characters as well as unicode emojis.
  • Slowmode: Slowmode, when enabled, will limit a user's message-rate. If it's set to 5 seconds, users can only send one message every 5 seconds. This can be quite useful for servers with lots of members.
  • NSFW: If a channel is dedicated to 18+ content, it's recommended to toggle this option as it will warn users before they join the channel.
Permissions, invites and webhooks are topics that will be discussed in further detail in their own sections.
